# javascript-DOM

A collection of practical examples demonstrating JavaScript DOM manipulation. Explore how to dynamically interact with HTML elements, modify content, and build interactive web pages.

## Contents

- **src/js/**: JavaScript files for DOM practice.
- **index.html**: Entry point for the demo page.
- **README.md**: Project overview and instructions.

## Features

- Hands-on JavaScript DOM manipulation techniques.
- Live demo: [https://natnaeleyuel.github.io/javascript-DOM/](https://natnaeleyuel.github.io/javascript-DOM/)
- Modular examples for easy experimentation and learning.

## Usage

Clone the repository and open `index.html` in your browser to see the DOM examples in action:

```sh
git clone https://github.com/natnaeleyuel/javascript-DOM.git
```
